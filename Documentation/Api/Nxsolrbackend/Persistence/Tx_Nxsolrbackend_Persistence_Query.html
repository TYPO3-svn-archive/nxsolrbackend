execute (line 67)

Executes the query. In contrast to Extbase's Query->execute() a QueryResult object is returned even if returnRawQueryResult is set to TRUE. This is necessary as a plain array of rows cannot contain for example facets and other additional information. If you need an array of objects, use the QueryResultInterface::toArray() method.
